# Public datasets
All experimental datasets used in this study are publicly accessible. The Human Lymph Node dataset is available from the Gene Expression Omnibus under accession number GSE263617. The Human Tonsil dataset co-profiling transcriptomes and proteins is available at https://www.10xgenomics.com/resources/
datasets/gene-protein-expression-library-of-human-tonsil-cytassist
-ffpe-2-standard. The ATAC-RNA-seq mouse brain multi-omics dataset is available through the UCSC Cell Browser (https://brain-spatial-omics.cells.ucsc.edu). The mouse embryonic MISAR-seq dataset is deposited in the National Genomics Data Center under accession number OEP003285. We further generated two simulated spatial multi-omics datasets following the nonnegative spatial factorization framework [1].
